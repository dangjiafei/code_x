# 在线互动病例创建流程实现计划

## 需求分析

用户核心需求：
1. **去掉隐私信息**：病例中不能包含姓名等隐私内容
2. **调整结构**：按照同事提供的结构调整设计（基本信息、入院检查、病理结果、诊断结果、治疗方案、治疗过程、随访结果）
3. **保持医学规范**：确保病例设计符合医学规范

## 活动规则设计

### 病例创建与投稿规则

| 规则 | 描述 |
|------|------|
| **草稿创建数量** | 无限制，用户可创建多个病例草稿 |
| **最终投稿数量** | 每个活动限投1份作品 |
| **草稿状态** | 支持保存、编辑、删除草稿 |
| **投稿状态** | 投稿后不可修改，进入审核流程 |
| **隐私保护** | 禁止包含患者姓名等隐私信息，支持匿名发布 |
| **审核结果** | 审核通过/驳回/待审核 |

### 病例创建流程

1. **报名活动** → 2. **创建草稿**（可多次）→ 3. **编辑优化** → 4. **提交审核**（1次/活动）→ 5. **审核通过** → 6. **发布上线**

## 功能模块实现

### 1. 隐私保护机制

```javascript
// 表单提交前的隐私检查
function checkPrivacy(formData) {
  const privacyKeywords = ['姓名', '身份证', '电话', '地址', '具体职业', '单位名称'];
  let hasPrivacyInfo = false;
  let detectedKeywords = [];
  
  // 递归检查所有字段
  function checkField(fieldValue, fieldName) {
    if (typeof fieldValue === 'string') {
      for (const keyword of privacyKeywords) {
        if (fieldValue.includes(keyword)) {
          hasPrivacyInfo = true;
          detectedKeywords.push(keyword);
        }
      }
    } else if (typeof fieldValue === 'object' && fieldValue !== null) {
      for (const [key, value] of Object.entries(fieldValue)) {
        checkField(value, key);
      }
    }
  }
  
  checkField(formData, 'root');
  
  if (hasPrivacyInfo) {
    showNotification(`病例中包含隐私信息：${detectedKeywords.join('、')}，请删除后重试！`);
    return false;
  }
  
  return true;
}
```

### 2. 草稿管理功能

```javascript
// 保存草稿功能
function saveDraft() {
  const currentStep = getCurrentStep();
  const formData = collectFormData(currentStep);
  
  // 隐私检查
  if (!checkPrivacy(formData)) {
    return;
  }
  
  const draftData = {
    id: generateDraftId(),
    title: formData.basicInfo?.title || '未命名病例',
    activityId: getCurrentActivityId(),
    userId: getCurrentUserId(),
    status: 'draft',
    createdAt: new Date().toISOString(),
    updatedAt: new Date().toISOString(),
    data: formData
  };
  
  // 保存到localStorage
  const drafts = JSON.parse(localStorage.getItem('caseDrafts') || '[]');
  const existingDraftIndex = drafts.findIndex(draft => draft.id === draftData.id);
  
  if (existingDraftIndex >= 0) {
    drafts[existingDraftIndex] = draftData;
  } else {
    drafts.push(draftData);
  }
  
  localStorage.setItem('caseDrafts', JSON.stringify(drafts));
  showNotification('草稿保存成功！');
}
```

### 3. 提交审核功能

```javascript
// 提交审核功能
function submitForReview(draftId) {
  // 检查是否已投稿
  const activityId = getCurrentActivityId();
  const submittedCases = JSON.parse(localStorage.getItem('submittedCases') || '[]');
  const hasSubmitted = submittedCases.some(caseItem => caseItem.activityId === activityId);
  
  if (hasSubmitted) {
    showNotification('您已提交过该活动的病例，每个活动限投1份作品！');
    return;
  }
  
  // 获取草稿数据
  const drafts = JSON.parse(localStorage.getItem('caseDrafts') || '[]');
  const draft = drafts.find(d => d.id === draftId);
  
  if (!draft) {
    showNotification('草稿不存在！');
    return;
  }
  
  // 隐私检查
  if (!checkPrivacy(draft.data)) {
    return;
  }
  
  // 提交审核
  const submittedCase = {
    id: generateCaseId(),
    draftId: draft.id,
    activityId: activityId,
    userId: getCurrentUserId(),
    status: 'pending',
    submittedAt: new Date().toISOString(),
    data: draft.data
  };
  
  // 保存到localStorage
  submittedCases.push(submittedCase);
  localStorage.setItem('submittedCases', JSON.stringify(submittedCases));
  
  // 更新草稿状态
  const draftIndex = drafts.findIndex(d => d.id === draftId);
  if (draftIndex >= 0) {
    drafts[draftIndex].status = 'submitted';
    localStorage.setItem('caseDrafts', JSON.stringify(drafts));
  }
  
  showNotification('病例已提交审核，请等待审核结果！');
  window.location.href = '/html/case-management.html';
}
```
