# CRM 管理后台 - 平台活动广场模块 PRD（页面说明版）

## 一、文档说明

本文档基于 CRM 管理后台现有页面梳理，用于描述“平台活动广场”模块的页面结构、展示内容、字段、筛选项、操作、弹窗和新建流程。

本文档不包含技术实现、接口定义、数据表结构、埋点方案等技术内容。

## 二、模块定位

平台活动广场用于 CRM 管理后台统一创建、管理、发布和查看平台活动。平台创建的活动可被厂家端使用，后续可在 CRM 后台查看活动参与明细、医生投稿状态、内容传播效果、调研评估和操作记录。

模块同时承接厂家端提交的客户需求反馈，包括新活动需求和已有活动的主题建议。

## 六、新建活动页

### 6.1 页面目标

新建活动页用于平台管理员创建平台活动。页面采用分步骤流程，帮助管理员逐步填写基础信息、征稿主题、投稿要求、活动概览和活动海报。

### 6.2 顶部操作

| 操作 | 说明 |
| --- | --- |
| 返回列表 | 返回活动列表页。 |
| 保存草稿 | 保存当前填写内容为草稿。 |
| 发布活动 | 校验必填内容后发布活动，并返回活动列表页。 |

### 6.3 步骤条

新建活动共 4 步：

| 步骤 | 名称 | 说明 |
| --- | --- | --- |
| 第 1 步 | 基础信息 | 填写活动基本配置。 |
| 第 2 步 | 征稿主题与投稿要求 | 配置内容类型下的征稿主题和通用投稿要求。 |
| 第 3 步 | 活动概览生成 | 生成和编辑医生端展示的活动概览。 |
| 第 4 步 | 活动海报 | 生成活动封面海报并完成发布。 |

步骤操作：

| 操作 | 说明 |
| --- | --- |
| 下一步 | 校验当前步骤必填项，通过后进入下一步。 |
| 上一步 | 返回上一流程步骤。 |
| 保存并发布 | 在第 4 步完成活动发布。 |

### 6.4 第 1 步：基础信息

字段：

| 字段 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| 医学领域 | 下拉选择 | 是 | 选择活动所属医学领域。 |
| 活动名称 | 文本输入 | 是 | 最多 50 字，不可与已有活动重名。 |
| 活动背景 | 多行输入 | 否 | 描述活动背景，例如领域现状、问题、开展契机。 |
| 活动目的与目标 | 多行输入 | 否 | 描述活动目的和可量化目标。 |
| 活动封面 | 图片上传 | 是 | 支持 jpg/png，比例 16:9，大小不超过 2MB。 |
| 招募时间范围 | 日期范围 | 是 | 选择开始日期和结束日期。 |

医学领域下拉选项：

| 选项 |
| --- |
| 心血管疾病 |
| 神经内科 |
| 骨科 |
| 消化内科 |
| 内分泌科 |
| 呼吸内科 |
| 新增领域 |

校验规则：

| 规则 | 提示 |
| --- | --- |
| 未选择医学领域 | 请选择医学领域 |
| 未填写活动名称 | 请输入活动名称 |
| 活动名称重复 | 该活动名称已存在，请更换名称 |
| 未选择开始日期 | 请选择开始日期 |
| 未选择结束日期 | 请选择结束日期 |
| 开始日期早于当天 | 开始日期不能早于今天 |
| 结束日期早于开始日期 | 结束日期不能早于开始日期 |
| 上传图片超过 2MB | 图片大小不能超过 2MB |

### 6.6 第 2 步：征稿主题与投稿要求

页面分为“征稿主题”和“投稿要求”两块。

#### 征稿主题

按内容类型分组配置主题：

| 内容类型 | 说明 |
| --- | --- |
| 医学播客 | 适合经验复盘、诊疗思路、专题讲述。 |
| 手术/操作视频 | 适合操作流程、关键步骤、并发症处理。 |
| 在线互动病例 | 适合病例推演、诊疗选择、互动问答。 |

每个内容类型卡片展示：

| 内容 | 说明 |
| --- | --- |
| 类型名称 | 如医学播客、手术/操作视频、在线互动病例。 |
| 类型说明 | 展示该内容类型适合的创作方向。 |
| 主题数量 | 展示当前类型下已添加主题数量。 |
| 展开/收起按钮 | 控制当前类型主题配置区域显示或隐藏。 |
| 新增主题按钮 | 在当前内容类型下新增一个主题。 |

主题字段：

| 字段 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| 主题名称 | 文本输入 | 是 | 最多 30 字。 |
| 主题说明 | 多行输入 | 是 | 说明医生围绕什么临床方向创作，最多 200 字。 |
| 创作重点 | 文本输入 | 否 | 例如诊断依据、治疗取舍、随访管理。 |

主题操作：

| 操作 | 说明 |
| --- | --- |
| 新增主题 | 在对应内容类型下新增主题。 |
| 上移 | 调整主题顺序。 |
| 下移 | 调整主题顺序。 |
| 删除 | 删除当前主题；至少保留 1 个主题。 |

限制与校验：

| 规则 | 提示 |
| --- | --- |
| 每种类型最多添加 10 个主题 | 每种类型最多只能添加10个主题 |
| 删除至无主题 | 至少需要保留1个主题 |
| 主题名称或主题说明为空 | 请填写所有主题的名称和描述 |

#### 投稿要求

投稿要求为所有主题共用的底线要求。

默认要求：

| 标签 | 内容 |
| --- | --- |
| 内容完整 | 请说明病例背景、诊断依据、治疗过程、关键判断和复盘思考。 |
| 隐私保护 | 请确保患者姓名、身份证号、联系方式、影像编号等隐私信息已脱敏。 |
| 学术价值 | 请突出诊疗思路、决策依据、经验启发或特殊病例特点。 |

字段：

| 字段 | 说明 |
| --- | --- |
| 标签 | 投稿要求标题，如内容完整。 |
| 详细内容 | 投稿要求说明。 |

操作：

| 操作 | 说明 |
| --- | --- |
| 新增 | 新增一条投稿要求。 |
| 删除 | 删除当前投稿要求；首条默认要求不可删除。 |

校验：

| 规则 | 提示 |
| --- | --- |
| 没有任何投稿要求 | 请至少填写一条内容要求 |

#### 内容类型建议时长

| 内容类型 | 建议时长选项 |
| --- | --- |
| 医学播客 | 15-20 分钟、20-40 分钟 |
| 手术/操作视频 | 5-10 分钟 |

活动头部展示：
